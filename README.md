# begegnungserfassung

Vue application for https://github.com/fiedsch/contao-ligaverwaltung-bundle (Contao Backend data entry)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Library build

```
npm run lib
```
